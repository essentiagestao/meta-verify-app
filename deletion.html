Exclusão de Dados

Esta página descreve como solicitar a exclusão de dados relacionados ao uso do Meta Ads Manager pelo Grupo Saga.

Quais dados são armazenados?

O aplicativo não armazena dados pessoais de usuários finais. Os únicos dados retidos são tokens de acesso à API do Meta, armazenados localmente nas máquinas dos administradores autorizados.

Como solicitar exclusão

Para solicitar a exclusão de qualquer dado associado ao uso deste aplicativo, entre em contato por e-mail:

estevaopbxs@gmail.com

Inclua no e-mail: nome, empresa e descrição dos dados a serem excluídos. A solicitação será processada em até 30 dias.

Revogação de acesso

Para revogar o acesso do aplicativo à sua conta Meta, acesse Configurações do Facebook → Aplicativos e sites e remova o aplicativo da lista.
